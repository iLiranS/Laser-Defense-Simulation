
# Laser Defense Simulation

An interactive 3D visualization of a laser-based defense system, where interceptors detect and neutralize incoming missiles on a simulated globe. (WIP)

## Features

- [x] Placing objects on the sphere
- [x] drawing curves between 2 points with animations
- [x] Atmosphere shader + post processing
- [ ] Objects following a curve
- [ ] Models for the project
- [ ] Interceptor w/ systems
- [ ] Physics based calculations
- [ ] Friendly UI for interactions
- [ ] State management, better visuals, performance improvements, scheduling system, much more...

## Tech Stack

Three.js + React Three Fiber (R3F) + Drei + Zustand + TailwindCSS
